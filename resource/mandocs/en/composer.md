# composer

composer is php package manager tool.

## install

```bash
brew install composer
```

## install package

examples:

```bash
composer install inhere/console
```

## aliyun repos

`composer.json` config:

```json
"repositories": {
  "packagist": {
    "type": "composer",
    "url": "https://mirrors.aliyun.com/composer/"
  }
}
```
